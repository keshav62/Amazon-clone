Amazon Clone Website
This project is a fully functional Amazon clone built using HTML and CSS. The goal of this project is to replicate the user interface of the Amazon website, providing a comprehensive learning experience in web development, focusing on HTML and CSS.

Features
Responsive Design: The website is fully responsive and works seamlessly on different devices, including desktops, tablets, and mobile phones.

Home Page: Features a variety of products, promotional banners, and navigational links similar to the actual Amazon website.

Product Listings: Individual pages for different product categories with detailed product descriptions.

Search Functionality: A search bar to help users find products quickly.

Navigation Bar: A fully functional navigation bar with links to different sections of the website.

Footer: An informative footer section with links to various informational pages.

Technologies Used
HTML: The structure of the website is built using HTML, providing a semantic and accessible framework.

CSS: Used for styling the website, ensuring a visually appealing and user-friendly interface.
